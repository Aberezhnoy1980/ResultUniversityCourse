Вам дан массив чисел:

        const numbers = [10, 4, 100, -5, 54, 2];
        Необходимо посчитать сумму кубов всех чисел в массиве. Например, для массива [1, 5, 9] должен 
        быть следующий результат: 1^3 + 5^3 + 9^3 = 1 + 125 + 729 = 855

        Решите данную задачу 4-мя способами:

        Через цикл for
        Через цикл for...of
        Через метод forEach()
        Через метод reduce()
        Для данного массива numbers сумма кубов должна равняться 1158411.
