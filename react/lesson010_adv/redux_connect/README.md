# React Redux connect()

* <em style="color: #EB5757">connect()</em> — функция, которая подключает компонент React к хранилищу Redux. Данный метод актуален в версиях React младше 16.8, в новых версиях не рекомендуется использовать <em style="color: #EB5757">connect()</em>.
* Актуальные способы взаимодействия с хранилищем — <em style="color: #EB5757">useSelector()</em> и <em style="color: #EB5757">useDispatch()</em>.
* <em style="color: #EB5757">connect()</em> принимает коллбеки, которые определяют, какие данные из состояния (state) следует передавать в пропсы компонента и какие действия (actions) следует отправлять в ответ на определенные события.
* Использование <em style="color: #EB5757">onnect()</em>c усложняет код компонента, делает его менее тестируемым и вводит путаницу с пропсами.
